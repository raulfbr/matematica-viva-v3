1.  **Preparação**:
    -   Leia `_Assistente_Pessoal/RaulPessoal/06_CONSELHO_SABEDORIA_V3.md` para entender as personas.
    -   Leia `_Assistente_Pessoal/RaulPessoal/07_SISTEMA_OPERACIONAL_CONSELHO.md` para entender os protocolos.

2.  **O Dilema**:
    -   Identifique o dilema ou tensão central na fala do usuário.
    -   *Nota:* Se o usuário não forneceu detalhes, infira pelo contexto ou use o último tema ativo.
    -   **NÃO FAÇA PERGUNTAS DE DIAGNÓSTICO AGORA.** Guarde-as para o final.

3.  **A Távola Redonda (Debate Completo & Autônomo)**:
    -   **O Convite Expandido:**
        -   Adicione **UM Especialista Técnico/Business de Elite** relevante ao tema (Ex: _James Clear_ para Hábitos, _Cal Newport_ para Foco, _Peter Thiel_ para Zero to One, _Huberman_ para Biologia).
        -   Mantenha a Mesa Base: **Seth Godin** (Tribo), **Lewis** (Significado), **Mason** (Método).
    -   **O Debate (A Arena):**
        -   O confronto deve ser real. Deixe o Especialista Técnico atacar o "Romantismo" de Lewis/Mason.
        -   Busque a "Verdade do Meio" (A Lógica que serve ao Espírito).

4.  **Veredito Soberano**:
    -   Cristalize a sabedoria em 3 passos práticos e *imediatos*.

5.  **Materialização (Registro Obrigatório)**:
    -   Gere IMEDIATAMENTE o relatório final usando `write_to_file`.
    -   **Caminho:** `_Assistente_Pessoal/RaulPessoal/xRelatorios_Conselho/`
    -   **Nome:** `YYYY-MM-DD_[TEMA_CHAVE].md`
    -   **Conteúdo do Relatório (Estrutura Fria):**
        -   **1. A Arena (Expanded):** Resumo dos argumentos de alto nível, destacando o choque entre a Técnica (Novo Especialista) e a Alma (Base).
        -   **2. O Veredito:** Ação prática, seca e direta.
        -   **3. O Tesouro (Visual & Impacto):**
            -   **Frases de Efeito:** 3 quotes curtas e brutais extraídas do debate.
            -   **Mindset:** Atmosfera necessária.
        -   **4. O Aprendizado (A Síntese):**
            -   **Conceito-Chave:** Um termo técnico ou filosófico para o usuário aprender/fixar (Ex: "Ativação Comportamental", "Metanoia").
            -   **Cristocentrismo:** Como isso aponta para Cristo no final.
    -   **Retorno ao Usuário:** Apresente apenas as *Frases de Efeito* e o link para o arquivo.
